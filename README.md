# 가중 코호트 트렌드를 활용한 헬스케어 보험 동적 프라이싱 모델
> "단순한 질병 유무가 아닌, '악화 속도'에 집중하여 보험사의 재무 건전성과 고객의 건강을 동시에 지키는 데이터 기반 프라이싱 모델"

<br>

## 1. 프로젝트 배경 및 문제 정의
* **시장 한계:** 기존 유병자 보험은 획일화된 리스크 평가(예: 3-5-5 질문지)로 인해 모든 유병자에게 일괄적인 할증을 부과하며, 이는 고위험군의 역선택과 막대한 수준의 적자 유발
* **타겟 질환:** 막대한 투석 치료 비용이 발생하는 '당뇨 신장병증' (공복혈당 126 이상 $\rightarrow$ 사구체여과율 60 미만 전환군)
* **목표:** 과거 이력이 아닌 시계열적 '악화 속도(Trend)'를 반영한 동적 프라이싱과 보장 체증형 설계를 통해 선제적 리스크 관리 수행

<br>

## 2. 사용 기술 및 데이터
* **Language/Libraries:** `Python`, `Pandas`, `NumPy`, `LightGBM`, `SHAP`, `Seaborn`
* **Dataset:** 국민건강보험공단 건강검진 데이터 3개년 (2022~2024, 각 100만 건 샘플링)
    * 개인 식별이 불가능한 공공데이터의 한계를 극복하기 위해 성별/연령대/지역 기반의 **'가상 코호트(Pseudo-Cohort)'** 생성
    * 연령 이동(Aging-in/out)을 고려한 80:20 가중 평균으로 또래 대비 악화 속도(Trend) 파생 변수 추출

<br>

## 3. 핵심 분석 파이프라인

### Step 1. 모델링 및 설명 가능한 AI (XAI) 도입
* 대규모 정형 데이터 처리에 최적화된 **LightGBM**을 활용하여 신장병증 전환 위험도 예측
* **SHAP** 알고리즘을 도입하여 금융/의료 도메인의 핵심 과제인 '모델의 판단 근거(블랙박스 해소)' 시각화 및 주요 임상 요인 검증 수행

### Step 2. 비용 최적화 (KneeCurve Optimization)
* 놓친 위험 비용(FN)과 예방 관리 비용(FP) 간의 Trade-off를 수식화하여 총비용 시뮬레이션 수행
* 비즈니스 효율이 극대화되는 최적의 집중 관리 타겟팅 범위를 상위 7%로 도출

### Step 3. 보장 체증형 동적 프라이싱 산출
* 예측된 전환 위험도를 기반으로 고객을 3등급(Grade A, B, C)으로 세분화
* 고위험군에게는 초기 보장을 제한(500만 원)하여 역선택 방어, 밀착 헬스케어(CGM 제공 등) 후 수치 개선 시 보장액을 상향하는 선순환 구조 설계
* Z-Score 표준화를 통해 개별 고객에게 보험료가 오른 이유를 객관적인 수치로 설명하는 맞춤형 앱 푸시 알림 로직 구현

<br>

## 4. 비즈니스 임팩트
* **관리 예산 절감:** 전체 대상자 일괄 관리 대비, 최적 타겟팅(상위 7%)을 통해 예방 관리 예산 **74% 절감** (155억 $\rightarrow$ 39.8억)
* **재무 건전성 확보 (1년 차 시뮬레이션):** 기존 획일적 보장 시 발생하던 1,130억 원의 적자를 영업이익률 39.8%에 달하는 **연간 184억 원 흑자로 전환**(공공데이터 샘플링 및 추정 단가 기반의 가상 시뮬레이션 기준)
